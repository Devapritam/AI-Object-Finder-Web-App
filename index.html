AI Object Finder Web App

NOTE: To use this app, first you have to type a desired object name and click on start. Then the app will start detecting the objects in webcam mode and if the object that you mentioned is found by the model, then the model will speak out "'${Object Name}' found", else the model will show "'${Object}' not found"
